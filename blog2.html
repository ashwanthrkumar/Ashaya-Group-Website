Ways to Whatsapp message without saving contact

Want to Whatsapp message a person without saving contact number. Here are 2 ways..

    Via wa.me

  • Open browser, type wa.me/91[mobile number]
  • Example: if the number is 9876543210 then wa.me/919876543210

    Via WhatsDirect App

  • Download WhatsDirect Application from Google PlayStore
  • Under Enter number here, enter the number to be messaged.
